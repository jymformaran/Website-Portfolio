OBJECTIVE

The goal of this project is to develop an enhanced school supplies vending machine that provides a convenient one-stop solution for students and staff. By integrating sanitary items, a modern touchscreen interface, an add-to-cart system, and a status light indicator, the machine is designed to improve accessibility, streamline purchasing, and simplify maintenance while delivering a more efficient and user-friendly experience.

ABOUT

This project modernizes the traditional school supplies vending machine by combining educational essentials and sanitary products into a single automated kiosk. Built with a touchscreen interface and multi-item purchasing capability, it allows users to complete transactions quickly and conveniently. The system also features status indicators for easier monitoring and maintenance, accepts multiple denominations of Philippine currency, and was designed with reliability, usability, and operational efficiency in mind. Overall, the project demonstrates how technology can improve access to essential items while enhancing the user experience within an educational environment.
