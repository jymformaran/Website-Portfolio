OBJECTIVE

Develop a mobile application as a course requirement that enhances students' mathematical skills through timed quizzes while providing a secure and engaging learning experience using modern mobile development technologies.

ABOUT

Smart Vee is a mobile application developed as a school project to fulfill the requirements of a mobile application development subject. The app offers timed mathematics quizzes in an interactive and user-friendly environment, encouraging users to improve their speed and accuracy in solving problems. It integrates Firebase for secure user authentication and data management, demonstrating practical experience in mobile development, cloud services, and responsive user interface design.
