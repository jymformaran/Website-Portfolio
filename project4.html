OBJECTIVE

Develop a responsive web portal for the A.C.T.S. organization as a course requirement, providing secure authentication, profile management, and efficient data administration while applying full-stack web development concepts and database integration.

ABOUT:

The A.C.T.S. School Organization Portal is a school project created to fulfill the requirements of a web development subject. It is a responsive full-stack application built using Django, MySQL, and modern web technologies, featuring secure user authentication, profile management, and complete CRUD (Create, Read, Update, Delete) functionality. The system was designed to streamline organization management, improve accessibility for members, and demonstrate practical skills in both front-end and back-end development.
